Redesign the landingpage matching to the invite theme 

Design and implement a premium landing page for a Digital Invitation Platform inspired by the structure, elegance, cinematic feel, spacing, transitions and premium experience of Lumivows, but DO NOT copy any copyrighted assets, layouts pixel-for-pixel, branding, illustrations or imagery.

The landing page should visually match the invitation theme used throughout this platform:
- Warm ivory background (#F9F5EE)
- Champagne gold accents
- Soft beige gradients
- Elegant serif typography
- Luxurious minimal aesthetic
- Light theme only
- Premium wedding stationery feel
- Large white spaces
- Soft shadows
- Frosted glass cards where appropriate
- Thin decorative dividers
- Subtle floral/ornamental decorations

The page should feel like a luxury invitation rather than a SaaS dashboard.

GENERAL DESIGN LANGUAGE
=================================================

Overall experience:
- Minimal

- Cartoon illustrations
- Busy layouts
- Rounded playful designs

Everything should feel like opening a luxury invitation.

=================================================
SCROLL BEHAVIOUR
=================================================

Desktop:
- Implement Lenis smooth scrolling
- Very smooth momentum
- No lag
- Hardware accelerated
- Disable if user prefers reduced motion

Mobile:
- Disable Lenis
- Use native browser scrolling
- Ensure scrolling remains perfectly smooth

Implementation should automatically detect device width.
